== Description ==

Factlink works on helping people understand how credible the information is that they’re reading online.

This plugin enables the Factlink functionality on your Wordpress site.

== Installation ==

1. Upload the file `factlink.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done! You should now be able to use Factlink.
